'File It Up' Data Privacy

Camera use

In the app 'File It Up', the phone camera will be used to take pictures of the documents the user wants to store.

External Services

OCR.space

To recognize the text of the document, the external service 'OCR.Space' is used. (https://api.ocr.space)

This service needs to receive a picture of the document. This picture will not be stored beyond this use and the generted PDF will be deleted 60 minutes after use.

More information: https://ocr.space/privacypolicy

Img-Resize.com

In Order to send the picture of the document to the ocr service, the picture must be resized. For this, the service img-resize.com is used.

This service will also receive the picture of the document.

Microsoft OneDrive

The purpose of this app is to finally upload the PDF to the user's OneDrive cloud storage.

In order to do so, the app will need the user to log in to OneDrive on the microsoft web site.

In order to store files to the user's storage, the user needs to give permission to read and write to the OneDrive App-Folder.
This is a special folder seperated from all other files and folders on the user's OneDrive storage, 'File It Up' will never have access to any other files outside of that folder.
Neither the login credentials, nor any of the user's OneDrive data will ever be stored outside of the user's phone's storage.
